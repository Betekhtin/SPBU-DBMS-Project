# SPBU-DBMS-Project
Project for Database Management Systems course in Saint-Petersburg State University  
Web-service for weather forecasting and viewing meteorological history
##Data structure
The database will contain data for cities, years, weather and user data
##Goal
Application should be able to make predictions about weather on a certain day based on previous meteorological data, display weather history for determined time-period and possibly look for weather anomailes.
